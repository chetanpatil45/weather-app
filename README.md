# Weather application 

### Tech Stack 
- **Language**: Java
- **Framework**: Spring Boot,
- **API**:  OpenWeatherAPI
- **Frontend**: Thymeleaf, HTML5, CSS3, Javascrit
- **Server**: Apache Tomcat 10.1
- **Build Tool**: Maven
- **Concepts**: `RestTemplate`, `IoC`, Depedency Injection, **RESTful API's**
